## Droid-Guide works on android 24+ and it uses the following
---

- Material Design & ConstraintLayout
- View Binding & Data binding
- Coroutines & Kotlin Flow
- Android Hilt
- Multi module Clean MVVM Architecture
- Navigation Graph
- Mockito & JUnit4
- Retrofit
- Image loading using Picasso
- etc
